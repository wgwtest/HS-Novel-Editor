# 验收大纲

## 当前验收范围

当前验收范围是 `P0.1 文档治理底座`，不是页面功能验收。

## 验收项

| 编号 | 验收项 | 方法 |
| --- | --- | --- |
| `A-P0.1-01` | 正式文档根存在 | 检查 `DOC/CODEX_DOC/README.md`。 |
| `A-P0.1-02` | 文档分类完整 | 检查 `01_需求分析` 至 `08_原型与附图`。 |
| `A-P0.1-03` | 事实源索引明确 | 检查 `02_设计说明/00-设计事实源索引.md`。 |
| `A-P0.1-04` | 迁移规则明确 | 检查 `03_规范与流程/00-文档治理与迁移规则.md`。 |
| `A-P0.1-05` | WBS 和节点合同存在 | 检查 `04_研发计划/` 和 `05_节点合同/`。 |
| `A-P0.1-06` | `验证作品/` 不入库 | 检查 `.gitignore` 和 `git status --ignored`。 |
| `A-P0.1-07` | 原型运行基线未破坏 | 执行当前基准原型的校验脚本。 |
