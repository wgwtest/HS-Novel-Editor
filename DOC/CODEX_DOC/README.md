## 协作规则

- 新的稳定工程文档默认写入 `DOC/CODEX_DOC/`。
- 现有 `设计说明/` 和 `原型包/` 暂作为历史事实源和资产根保留，不在本轮移动。
- `验证作品/` 只作为本地素材区，必须保持在 `.gitignore` 中。
- 页面和数据改动必须先确认事实源属于对象定义、投影生成、页面绘制还是视图状态。
- 当前基准页面仍以 `原型包/2026-06-22-叙事验证工具-基准时间轴原型-v1/source/index.html` 为实现入口。
